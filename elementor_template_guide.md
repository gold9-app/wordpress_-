# Elementor 템플릿 준비 가이드

## 1. WordPress에서 템플릿 생성

1. **WordPress 관리자** → **템플릿** → **테마 빌더** → **단일 게시물**
2. 새 템플릿 생성 (예: "건강 칼럼 템플릿")
3. Elementor로 다음 섹션 디자인:

### 추천 구조:
```
┌─────────────────────────────┐
│ 헤더 섹션 (배경 이미지/그라디언트)│
│ - 제목                       │
│ - 부제목/요약                 │
│ - 저자 정보                   │
└─────────────────────────────┘
┌─────────────────────────────┐
│ 콘텐츠 영역 **[중요]**         │
│ - HTML 위젯 삽입              │
│ - 또는 동적 태그 사용          │
└─────────────────────────────┘
┌─────────────────────────────┐
│ CTA 섹션                     │
│ - 저자 소개                   │
│ - 인스타그램 링크             │
│ - 관련 글 추천                │
└─────────────────────────────┘
```

## 2. 템플릿 ID 확인

템플릿을 저장한 후 **URL에서 template ID**를 확인하세요.
예: `post.php?post=1234&action=elementor` → ID는 **1234**

## 3. 템플릿 조건 설정

- 특정 카테고리(ID: 22)에만 적용
- 또는 모든 글에 적용

---

## 방법 A: HTML 위젯 활용 (간단)

**장점**: 기존 HTML을 그대로 사용 가능
**단점**: Elementor 스타일 제한적

### 템플릿 구조:
1. 헤더 섹션 (Elementor로 디자인)
2. HTML 위젯 (여기에 `{{CONTENT}}` placeholder)
3. 푸터 섹션 (Elementor로 디자인)

---

## 방법 B: 동적 태그 활용 (추천)

**장점**: Elementor와 완전 통합, 가시성 최고
**단점**: HTML을 Elementor 위젯으로 변환 필요

### 구현:
1. Elementor 템플릿에 **Text Editor 위젯** 삽입
2. 동적 태그 → **Post Content** 선택
3. 스타일 커스터마이징 (폰트, 색상, 간격)

---

## 방법 C: Elementor JSON 직접 생성 (고급)

HTML을 파싱하여 Elementor JSON으로 변환하고 `_elementor_data` 메타필드에 저장

**장점**: 완전 자동화, 최고의 가시성
**단점**: 복잡한 구현
